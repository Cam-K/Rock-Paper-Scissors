What are the things you learned about in this course and their purpose?

We learned many things in this course. The main things we’ve learned in the course was how to write a website with HTML, style it with CSS and program with Javascript. In HTML, we learned how to use tags. Tags are what defines what on a webpage. For example, the p text was used for paragraphs. Some other tags we learned was the h1 tag for headers, the h2 for sub headers, and ol for ordered lists. CSS is used for styling and can target specific tags and ID’s in the html document for give it things like fonts, colors and bolded and underlined text. Some tags include color, background-color, and text-decoration. Lastly we learned about Javascript. Javascript is different from HTML as HTML is a markup language and does not contain any logic. Javascript is a programming language and contains logic. In the Javascript language we learned how to use alerts, prompts, set a variable, use if statements, while and for loops, session storage, get element by id, and forms. For example, alerts and prompts would open up a small window and have a box in them for you to type something. Also, forms were used to make things like text boxes and number selectors. Get element by id targets an id in the webpage and you can do various things with it like input your own text.
